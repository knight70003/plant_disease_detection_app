🌿 PlantDoc AI

PlantDoc AI is a Deep Learning-based intelligent plant disease detection and recommendation platform developed using Django and TensorFlow. The system utilizes a Convolutional Neural Network (CNN) trained on a Kaggle dataset containing 10 classes of healthy and diseased plant leaf images for automated disease classification.

The platform performs image preprocessing, feature extraction, confidence-based disease prediction, and AI-assisted recommendation generation for preventive agricultural guidance. In addition, the application includes secure authentication, prediction history management, and analytical visualization for disease monitoring.

---

🚀 Key Features

- CNN-based plant disease classification
- Real-time image prediction system
- AI-powered treatment recommendations
- Confidence score analysis
- Secure Login & Signup authentication
- Prediction history storage
- Analytics dashboard using Matplotlib & Seaborn
- Responsive Django-based web interface

⚙️ Workflow

1. User uploads plant leaf image
2. Image preprocessing is performed
3. CNN model predicts disease class
4. Confidence score is generated
5. AI recommendation is provided
6. Prediction data is stored in database
7. Analytics dashboard visualizes results

---

📊 Dataset

The CNN model is trained on a curated Kaggle plant disease dataset containing 10 categorized classes of healthy and diseased plant leaves.
